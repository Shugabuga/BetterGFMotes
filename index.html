BetterGFMotes

What is BetterGFMotes?

BetterGFMotes is a browser plug-in that allows you to view and use additional Gravity Falls-themed emotes on Reddit and Voat.

Features

  • Adds hundreds of new emotes to Reddit, and more to come!
  • See /r/gravityfalls emotes throughout Reddit
  • Search, sort, and view additional emotes
  • Adds emote modifiers, such as -r, -blur, -slide, and -sepia.
    • ]
  • Fully compatable with BPM's Chaos Script.
  • Adds site-wide spoiler tags
  • Works natively in many popular browsers

Download

You can install BetterGFMotes for Firefox by going here.

You can install BetterGFMotes on the Google Chrome Web Store by clicking here. This version also would also work on Yandex, Vivaldi, Opera (see below), and other Chromium-based browsers.

You can install BetterGFMotes with Opera if you first install the Download Chrome Extension plug-in for Opera, then installing the Google Chrome version above.
